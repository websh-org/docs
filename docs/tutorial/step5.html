Step 5 - Implement file operations

We will use a <textarea> as our text editor. Insert this into div#app:

  <textarea id="editor"><textarea>

Let's now implement the three file operations:

  const editor = document.getElementByID("editor");

  WebShellApp
  .command("file-new",()=>{
    editor.value = "";
  })
  .command("file-open",({content})=>{
    editor.value = content;
  })
  .command("file-save",()=>{
    return { content: editor.value };
  })

We are done!

You can now go to the WebShell Sandbox and open your app again. It will have the fool file toolbar, and you can actually use it to edit your text files. It's no worse than Windows Notepad!
